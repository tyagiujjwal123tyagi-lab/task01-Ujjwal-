# Personal Portfolio Website

## 📌 Project Overview

This project is a personal portfolio website developed using HTML, CSS, and JavaScript. It showcases my educational background, technical skills, projects, and contact information in a professional and user-friendly format.

The purpose of this portfolio is to create a strong online presence and provide recruiters and collaborators with a quick overview of my profile.

## 🎯 Objectives

- Showcase my skills and achievements.
- Present my projects in an organized way.
- Create a professional online portfolio.
- Improve my frontend web development skills.

## 🚀 Features

- Responsive design
- Modern user interface
- About Me section
- Skills section
- Projects section
- Contact section
- Interactive elements using JavaScript

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
